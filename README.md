What is JSON?

JSON is an abbreviation of Java Script Object Notation is a popular structure to store data in key and value pairs. They are enclosed by curly braces and it is the most widely
used format in NoSQL databases. JSON is language independent and can be used across multiple languages to interchange the data between two entities, mostly between the server and the client. MongoDB is a popular NoSQL database which uses JSON to store the data because of it's human readability and ease of use.
